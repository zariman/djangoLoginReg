To update database
    python manage.py makemigrations
    python manage.py migrate

To import User class
    from apps.user_login_ import *

To create a data using User class
    User.objects.create([field]=['value'])

Retrieve all users
    User.objects.all()

Get first user
    User.objects.first()

Get last user 
    User.objects.last()

Get users sorted by their first name
    User.objects.order_by("-first_name")

Get record of user id = 3 and change person's last_name
    x = User.objects.get(id=3)
    x.last_name = "Shawn"
    x.save()

Delete a record of user id=4
    User.objects.get(id=4).delete()
